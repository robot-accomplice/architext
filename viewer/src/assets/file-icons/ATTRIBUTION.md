# File-type icons

The technology brand glyphs in this directory are vendored from
[Devicon](https://github.com/devicons/devicon) (MIT License). Only the subset of
icons used by the Repo Tree view is included.

Monochrome-on-dark file types (JSON, Markdown, YAML, generic/config files) are
not taken from Devicon — they use Architext's own tinted glyphs in
`presentation/repoTreeFileIcons.js`, because the brand logos for those are
black/white and would be illegible on the dark viewer theme.
